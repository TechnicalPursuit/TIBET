Focus Test #10




This test sets up 4 'input' elements, all at the <body> element level.
None of the fields has a 'tabindex' attribute greater than -1, meaning that they're all programmatically controlled.
By pressing various keys, the focus can be navigated to them:

Home key navigates to the first field.
End key navigates to the last field.
Tab key navigates 'forward' to the next field and will wrap around.
Shift-Tab key navigates 'backward' to the previous field and will wrap around.

Because they're all at the <body> level (not part of a 'tibet:group'), navigation between these elements will automatically wrap.
