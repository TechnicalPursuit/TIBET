Relevant Test #1




This test sets up two XHTML elements, both bound to an underlying data model defined in the test code. One is a 'input type="checkbox"' element and the other is a 'input type="text"' element, which is made active or inactive dependent on whether the first control is checked or not. This 'relevancy' property of the underlying data model is modeled in the code. The attendant data types are also defined in the code.

The test code 'checks' the first element, which causes the bound data value to change and for the 'relevancy' of the text field to change. The text field changes its appearance based on that fact.

A variety of signals are thrown by the various objects involved.
