Data Test #10




This test sets up a <tibet:data> tag that makes data in the page available as under a particular named URI. It has 'buildout' set to 'true' so that when a 'set' is executed against the data that it is holding and there is no existing 'structure' for that data, that it is auto-created.
In this test, there is no pre-existing structure at all, which tests 'multi-level' structure building from a root.
A variety of signals are thrown by the various objects involved.
