Data Test #5




This test sets up a <tibet:data> tag that makes data in the page available as under a particular named URI. It has 'makestructures' set to 'true' so that when a 'set' is executed against the data that it is holding and there is no existing 'structure' for that data, that it is auto-created.
A variety of signals are thrown by the various objects involved.
