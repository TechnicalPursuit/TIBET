Drag Test #3




This test sets up a regular XHTML <div> element to be moved within a container.


The <div> element has a 'drag:mover' value of 'true' and a 'drag:container' value of '..', which means that it can be picked up from any place inside of the element itself (but not within its border) and dragged to any position, but only within its container element. The value of 'drag:container' should be some type of access path that can deal with markup (that is, a CSS selector path or XPath).
