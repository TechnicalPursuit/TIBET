Drag Test #2




This test sets up a regular XHTML <div> element to be resized.


The <div> element has a 'drag:resizer' value of 'true', which means that it can be resized from any side. Note that the mousedown to start this resizing must be *inside* of the element itself (within the red area - *not* the orange border) and within a certain number of pixels *inside* side. Since we did not specify the number of pixels of this 'perimeter' here, it defaults to 10.
