Drag Test #5




This test sets up 3 regular XHTML <div> elements to be dragged within a container.


The <div> element has a 'drag:mover' value of 'true' and a 'drag:item' value of '*', which means that any of the elements (as selected by the '*' path) can be moved but must stay within the containing element. The value of 'drag:item' should be some type of access path that can deal with markup (that is, a CSS selector path or XPath).
