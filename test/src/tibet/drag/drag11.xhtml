Drag Test #11




This test sets up a regular XHTML <div> element to be moved but locked to the X axis.


The <div> element has a 'drag:mover' value of 'true' and a 'drag:constraints' value of 'LOCK_X_TO_START_X', which means that it can be picked up from any place inside of the element itself (but not within its border) and dragged to any position within the <body> element. It's X dimension will be 'locked' and only the Y dimension will change.
