Drag Test #6




This test sets up a regular XHTML <div> element to be moved in 1.5em increments.


The <div> element has a 'drag:mover' value of 'true' and a 'drag:increment' of '1.5em', which means that it can be picked up from any place inside of the element itself (but not within its border) and dragged to any position within the <body> element. It will not move smoothly, but in 1.5em increments.
