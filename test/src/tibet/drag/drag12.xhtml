Drag Test #12




This test sets up 2 regular XHTML <div> elements by using a 'drag:info' attribute and another element as the place where the drag information is stored.


Both <div> elements have a 'drag:info' value of '#dragInfoSpan', which is the 'path' to the 'info' element - that is, the element that contains the drag information for any element that references it. The info element in this case specifies that any elements that reference it can be moved and will have an X offset of 1em and a Y offset of 1.5em and an X increment of 1.5em and a Y increment of 2em.
