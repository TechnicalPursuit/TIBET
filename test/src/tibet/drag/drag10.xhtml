Drag Test #10




This test sets up a set of regular XHTML <div> elements to be moved and resized from various sides with a variety of 'insets', showing that elements can be dragged or resized with either no inset or an inset that can be set 'per side'.

Drag from anywhere in a block within the rectangle, 10px inset from each side
Drag from anywhere in a block within the rectangle, 10px inset from the top and bottom sides and 5px inset from the left and right

Resize from anywhere in a block within the rectangle, 10px inset from each side
