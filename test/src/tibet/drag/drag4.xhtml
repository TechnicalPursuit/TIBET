Drag Test #4




This test sets up a regular XHTML <div> element to be resized within a container.


The <div> element has a 'drag:resizer' value of 'true' and a 'drag:container' value of '..', which means that it can be resized from any side but it's resizing must stay within the container element. Note that the mousedown to start this resizing must be *inside* of the element itself (within the yellow area - *not* the red border) and within a certain number of pixels *inside* side. Since we did not specify the number of pixels of this 'perimeter' here, it defaults to 10.
