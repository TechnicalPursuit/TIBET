Drag Test #7




This test sets up a regular XHTML <div> element to be moved with a 1em offset.


The <div> element has a 'drag:mover' value of 'true' and a 'drag:offset' of '1em', which means that it can be picked up from any place inside of the element itself (but not within its border) and dragged to any position within the <body> element. It will be offset by a 1em offset as it is dragged.
