XML Events Test #4




This test sets up two XMLEvents <ev:script> elements to listen for multiple events and multiple event observers. Note that these capabilities use TIBET extensions to the XMLEvents v2 specification.


This <ev:script> element has an 'event' value of 'TP.sig.DOMClick TP.sig.DOMDblClick' and an 'observer' containing 'fooDiv', meaning that either of those events coming from 'fooDiv' will cause an alert to be shown.
I'm the fooDiv. Click me or double click the mouse in me.




This <ev:script> element has an 'event' value of 'TP.sig.DOMClick' and an 'observer' containing 'barDiv bazDiv', meaning that the 'TP.sig.DOMClick' event coming from either the 'barDiv' or the 'bazDiv' will cause an alert to be shown.
I'm the barDiv. Click me.
I'm the bazDiv. Click me.
