XML Events Test #8




This test sets up an XMLEvents <ev:script> element to listen for keyboard sequences.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_A_Up__TP.sig.DOM_S_Up' (the TIBET way of observing the 'A' - shift a - key going 'up' followed by the 'S' - shift s - key going up) and an 'observer' containing '#document', meaning that the document should be the observer of those events.


This test sets up an XMLEvents <ev:script> element to listen for keyboard sequences using Unicode sequences.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_U0062_Up__TP.sig.DOM_S_Up' (the TIBET way of observing the 'b' key going 'up' followed by the 'S' - shift s - key going up) and an 'observer' containing '#document', meaning that the document should be the observer of those events.
