XML Events Test #5




This test sets up two XMLEvents <ev:script> elements to listen for events coming from anywhere and from any event observer. Note that these capabilities use TIBET extensions to the XMLEvents v2 specifiction.


This <ev:script> element has an 'event' value of '*' and an 'observer' containing 'fooDiv', meaning that ANY events coming from 'fooDiv' will cause an alert to be shown. Try clicking in it to get a TP.sig.DOMMouseDown event or scrolling the mouse wheel in it to get a TP.sig.DOMMouseWheel event.
I'm the fooDiv. Do something in me.




This <ev:script> element has an 'event' value of 'DOMClick' and an 'observer' of '*', meaning that the 'DOMClick' event coming from ANY source will cause an alert to be shown.
I'm the barDiv. Do something in me.
