XML Events Test #3




This test sets up two XMLEvents <ev:script> elements to listen for keyboard events.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_A_Up' (the TIBET way of observing the 'A' - shift A - key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_U0062_Up' (an alternate TIBET way of observing the 'b' key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_F2_Up' (the TIBET way of observing the 'F2' key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events.
