Service Test #10




This test sets up a <http:service> tag that services data at a particular URI. It shows how to perform a standard HTTP <form> POST using no parameter, but having a body (defined using a <tibet:data< tag in the head). It doesn't have a data type assigned, but will try to use a dynamically computed content type to hold the result data based on the data returned.
A variety of signals are thrown by the various objects involved.
