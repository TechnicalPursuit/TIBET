Bind Partial Attribute JavaScript Object - No Fragment




This test sets up a 'simple binding' between a textfield that is using an 'embedded expression' (which is whole attribute expression), a span that is using an 'embedded expression that is a partial expression and a piece of JavaScript Object data, set up by the <tibet:data> tag embedded in this document.


This test also uses a 'pure URI' as a top-level binding scope to avoid having to repeat the full qualifications for the bind scope on each bind.
