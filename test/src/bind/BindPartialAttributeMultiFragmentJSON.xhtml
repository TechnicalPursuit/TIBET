Bind Partial Attribute JSON - Multi-level Fragment




This test sets up a 'simple binding' between a textfield that is using an 'embedded expression' (which is whole attribute expression), a span that is using an 'embedded expression that is a partial expression and a piece of JSON data, set up by the <tibet:data> tag embedded in this document.


This test also uses a top-level binding scope of a 'pure URI' with a nested scope that contains a further qualifying fragment to avoid having to repeat the full qualifications for the bind scope on each bind.
