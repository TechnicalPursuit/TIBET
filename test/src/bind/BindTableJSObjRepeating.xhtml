Bind expressions in a table JavaScript Object




This test sets up a 'repeat' binding between the static Active Client Pages templating expressions in the page and a piece of JavaScript Object data, set up by the <tibet:data> tag embedded in this document. The repeating data rows and cells are automatically generated from the data, which is a collection.

Note that this test also includes variables in data expressions.
