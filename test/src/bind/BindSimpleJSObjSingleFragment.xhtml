Bind Simple JavaScript Object - Single Fragment




This test sets up a 'simple binding' between the various XHTML controls and a piece of JavaScript Object data, set up by the <tibet:data> tag embedded in this document.


This test also uses a URL with a 'top-level' fragment as a top-level binding scope to avoid having to repeat the full qualifications for the bind scope on each bind.
